## 题目描述
斐波那契数列是按如下递归定义的数列：

Fn = Fn-1 + Fn-2，且F1 = 1，F2 = 1。

因此斐波那契数列的前12项分别是：

F1 = 1

F2 = 1

F3 = 2

F4 = 3

F5 = 5

F6 = 8

F7 = 13

F8 = 21

F9 = 34

F10 = 55

F11 = 89

F12 = 144

在斐波那契数列中，第一个包含三位数字的是第12项F12。

在斐波那契数列中，第一个包含1000位数字的是第几项？
## 题解
很明显是大数加法，这里煮啵采用的的数组存数，倒序存储（即个位在数组首位）
## code
```C++
#include <bits/stdc++.h>
using namespace std;

int main(){
	ios::sync_with_stdio(0);
	cin.tie(0);
	cout.tie(0);
	vector<int> num1(1010,0);
	vector<int> num2(1010,0);
	num1[0]=1;
	num2[0]=1;
	int num=2;
	for(;;){
		vector<int> cmp(1010,0);//临时存储数组
		num++;//计数，是序列中的第几个
		int len;
		for(int i=0;i<1010;i++){
			int sum=cmp[i]+num1[i]+num2[i];//记得加上cmp[i]
			cmp[i]=sum%10;
			cmp[i+1]+=sum/10;
			if(cmp[i]!=0) len=i+1;//判断最高非零位，取得当前数位数
		}
		if(len>=1000) break;
		num1=num2;
		num2=cmp; 
	}
	cout<<num;
	return 0;
}
```
